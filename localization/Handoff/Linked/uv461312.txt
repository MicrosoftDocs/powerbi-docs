<h2 class="MsoNormal"><b><span style="font-family: Arial, sans-serif;">Task 4: </span></b><b><span style='color: rgb(54, 54, 54); font-family: "Arial",sans-serif; mso-fareast-font-family: "Times New Roman";'>Create additional visualizations using the
Report view </span></b></h2><p class="MsoNormal" style="line-height: normal; margin-bottom: 0pt; mso-margin-top-alt: auto; mso-outline-level: 2;"><b><span style='color: rgb(54, 54, 54); font-family: "Arial",sans-serif; mso-fareast-font-family: "Times New Roman";'></span></b><span style="font-family: Arial, sans-serif;">Now that we have
converted the data into the shape that we need for the rest of our analysis, we
can load the resulting table into our Report and create additional
visualizations.<o:p></o:p></span></p>

<p class="MsoNormal" style="line-height: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto;"><span style='color: rgb(54, 54, 54); font-family: "Arial",sans-serif; mso-fareast-font-family: "Times New Roman";'>Step 1: Load the query to your report<o:p></o:p></span></p>

<p class="MsoNormal" style="line-height: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto;"><span style='color: rgb(54, 54, 54); font-family: "Arial",sans-serif; mso-fareast-font-family: "Times New Roman";'>In order to load the query results to the
report, we can switch to the <b>Report view</b>,
using the view switcher at the bottom-left corner of the Power BI Designer
window.<o:p></o:p></span></p>

<p class="MsoNormal" style="line-height: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto;"><span><img src="https://powerbi.uservoice.com/assets/76050736/17.png"></span><br></p>

<p class="MsoNormal" style="line-height: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto;"><span style='color: rgb(54, 54, 54); font-family: "Arial",sans-serif; mso-fareast-font-family: "Times New Roman";'>This will trigger evaluation of the query and
load of the table output to the Report. You can check progress of this action
in the <b>Load</b> dialog.<o:p></o:p></span></p>

<p class="MsoNormal" style="line-height: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto;"><span><img src="https://powerbi.uservoice.com/assets/76050738/18.png"></span><br></p>

<p class="MsoNormal" style="line-height: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto;"><span style='color: rgb(54, 54, 54); font-family: "Arial",sans-serif; mso-fareast-font-family: "Times New Roman";'>After the query finishes loading, you will see
the updated fields in the <b>Fields pane</b>
at the right of the <b>Report view</b>.<o:p></o:p></span></p>

<p class="MsoNormal" style="line-height: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto;"><span style='color: rgb(54, 54, 54); font-family: "Arial",sans-serif; mso-fareast-font-family: "Times New Roman";'><o:p>&nbsp;</o:p></span></p>

<p class="MsoNormal" style="line-height: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto;"><span style='color: rgb(54, 54, 54); font-family: "Arial",sans-serif; mso-fareast-font-family: "Times New Roman";'>Step 2: Create a Line chart and a Bar chart<o:p></o:p></span></p>

<p class="MsoNormal" style="line-height: normal; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto;"><span style='color: rgb(54, 54, 54); font-family: "Arial",sans-serif; mso-fareast-font-family: "Times New Roman";'>In order to create a visualization, we can
drag fields from the <b>Field list</b> and
drop them in the <b>Report canvas</b>.<o:p></o:p></span></p>

<ol type="1" start="1">
 <li class="MsoNormal" style="color: rgb(54, 54, 54); line-height: normal; mso-list: l3 level1 lfo6; tab-stops: list 36.0pt; mso-margin-top-alt: auto; mso-margin-bottom-alt: auto;"><span style='font-family: "Arial",sans-serif; mso-fareast-font-family: "Times New Roman";'>Drag
     the <b>created_date</b> field and drop
     it in the <b>Report canvas</b>. This
     will create a new visualization in the <b>Report canvas</b>. In this case, it will also include the <b>shares</b> column since it’s a numeric
     value and create a <b>Line Chart</b>. <o:p></o:p></span></li>
</ol>
