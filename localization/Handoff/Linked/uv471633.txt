<a href="http://www.microsoft.com/en-us/server-cloud/solutions/business-intelligence/analysis.aspx" target="_blank">SQL Server Analysis Services</a>.&nbsp;Using state-of-the-art compression algorithms and
a multi-threaded query processor, the Analysis Services in-memory analytics engine delivers
fast access to tabular model objects and data by reporting client applications
such as Microsoft Excel and Microsoft Power BI.<br></span></font><br><font color="#2a2a2a" face="Segoe UI, sans-serif"><span style="line-height: 107%; font-size: 10pt;"><b>﻿</b></span><span style="line-height: 14.26px; font-size: 13px;"><b>High performance and highly secure</b></span></font><br><font color="#2a2a2a" face="Segoe UI, sans-serif"><div><span style="line-height: 14.26px; font-size: 13px;">Tabular model databases can scale to terabytes of data, partitioned for optimized processing.
