# Rename items in the Power BI service

[!INCLUDE [applies-no-desktop-yes-service](../includes/applies-no-desktop-yes-service.md)]

This article shows you how to rename a dashboard, report, report page, workbook, semantic model, app, and workspace in Power BI service.

**Can I change the name?**

| Content type | I'm the author or creator | Shared with me |
| --- | --- | --- |
| Dashboard in a workspace | :::image type="icon" source="../includes/media/yes-icon.svg" border="false"::: | :::image type="icon" source="../includes/media/no-icon.svg" border="false"::: |
| Report in a workspace | :::image type="icon" source="../includes/media/yes-icon.svg" border="false"::: | :::image type="icon" source="../includes/media/no-icon.svg" border="false"::: |
| Workbook in a workspace | :::image type="icon" source="../includes/media/yes-icon.svg" border="false"::: | :::image type="icon" source="../includes/media/no-icon.svg" border="false"::: |
| Semantic model in a workspace | :::image type="icon" source="../includes/media/yes-icon.svg" border="false"::: | :::image type="icon" source="../includes/media/no-icon.svg" border="false"::: |
| Workspace | :::image type="icon" source="../includes/media/yes-icon.svg" border="false"::: If you're the owner or have Admin permissions | :::image type="icon" source="../includes/media/no-icon.svg" border="false"::: |
| Published apps | :::image type="icon" source="../includes/media/yes-icon.svg" border="false"::: Not from the App screen, but the app name can be changed from the workspace and republished with a new name if you have Admin permissions | :::image type="icon" source="../includes/media/no-icon.svg" border="false"::: |
| App content (dashboard, report, workbook, semantic model) | :::image type="icon" source="../includes/media/yes-icon.svg" border="false"::: Not from the App screen, but the app's content can be renamed from the workspace and republished with a new name if you have Admin permissions | :::image type="icon" source="../includes/media/no-icon.svg" border="false"::: |
| Content in **Shared with me** | :::image type="icon" source="../includes/media/no-icon.svg" border="false"::: | :::image type="icon" source="../includes/media/no-icon.svg" border="false"::: |

## Rename a dashboard, report, or workbook

1. In a workspace, hover over the item to rename, select **More options** (...), and then select **Settings**.

   :::image type="content" source="media/service-rename/power-bi-workspace-dashboards-2.png" alt-text="Screenshot of My workspace in the Power BI service.":::

1. On the Settings page, type the new name and select **Save**.

   :::image type="content" source="media/service-rename/power-bi-rename-dashboard-3.png" alt-text="Screenshot of the Settings for Getting Started in Power BI dialog.":::

## Rename a semantic model

1. Start in a workspace and select the **Semantic models + dataflows** tab.
1. Hover over the semantic model to rename, select **More options** (...), and choose **Rename**.

    :::image type="content" source="media/service-rename/power-bi-rename-datasets-2.png" alt-text="Screenshot of My workspace in the Power BI service, highlighting Semantic models and dataflows.":::

    > [!NOTE]
    > The options in the dropdown vary.

1. In the Rename dialog, type a new name and select **Save**.

     :::image type="content" source="media/service-rename/power-bi-rename-2.png" alt-text="Screenshot of the Rename Financial Sample dialog.":::

## Rename a workspace

Anyone with Admin permissions can rename a workspace.

1. Start in the list of workspaces.
1. Next to the workspace you want to rename, select **More options** (...), and select **Workspace settings**. If you don't see this option, you don't have permissions to rename the workspace.

    :::image type="content" source="media/service-rename/power-bi-edit-workspace-2.png" alt-text="Screenshot that shows Workspace settings in the Workspaces list.":::

1. Type a new workspace name and select **Save**.

   :::image type="content" source="media/service-rename/power-bi-workspace-rename-2.png" alt-text="Screenshot of the Settings menu in the Power BI service.":::

## Rename a page in a report

Don't like the name of a page in your Power BI report? A new name is just a click away. Rename pages in [report Editing view](service-interact-with-a-report-in-editing-view.md).

1. Open the report in [Editing View](../explore-reports/end-user-reading-view.md).
1. Find the report page tabs at the bottom of the Power BI window.

    :::image type="content" source="media/service-rename/report-page-tabs-new.png" alt-text="Screenshot of the report page tabs at the bottom of the Power BI window.":::

1. Select the tab to open the report page that you want to rename.
1. Double-click the name on the tab to highlight it.

    :::image type="content" source="media/service-rename/hilite-tab.png" alt-text="Screenshot of the report tabs with a current tab name highlighted.":::

1. Type a new report page name and select **ENTER**.

    :::image type="content" source="media/service-rename/new-name.png" alt-text="Screenshot of the report tabs with a new tab name entered.":::

## Considerations and troubleshooting

- If you shared the item to rename, or if the item is part of an app, you won't see the gear icon and you won't have access to **Settings**.
- On the **Semantic models** tab, if you don't see **More options** (...), expand your browser window.
