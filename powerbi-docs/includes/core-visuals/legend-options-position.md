##### Position

With this dropdown menu, users have several options to define the position of the legend:
- **Top left**
- **Top center**
- **Top right**
- **Top left stacked**
- **Top right stacked**
- **Center left**
- **Center right**
- **Bottom left**
- **Bottom center**
- **Bottom right**
