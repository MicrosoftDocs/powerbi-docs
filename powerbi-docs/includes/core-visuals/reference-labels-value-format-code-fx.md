###### Format code

Where the selected label contains a data field that is numerical data, the **Format code** option allows you to specify custom format codes (based on Excel format codes)Â or apply conditional formatting. This option is available only when selecting custom display units.  

Where the selected label contains a data field that is date/time data, the **Format code** option allows you to select one of seven possible date formats.
