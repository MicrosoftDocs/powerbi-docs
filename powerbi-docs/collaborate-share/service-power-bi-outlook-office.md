# Use Power BI in Microsoft Outlook and Microsoft 365

You can use the Power BI app in Microsoft Outlook and Microsoft 365. By using this integration, you can access and work with Power BI content directly within Outlook and Microsoft 365 applications.

> [!NOTE]
> Looking for Power BI integration with Microsoft Teams? See [Collaborate with Power BI in Microsoft Teams](service-collaborate-microsoft-teams.md).

## Installation

If you install the Power BI app in Microsoft Teams, you also install it in Outlook and Microsoft 365. Otherwise, you can install the Power BI app from the store in Outlook and Microsoft 365. When you install the app, it also appears in Teams. The Power BI app in Teams continues to be generally available.

The Power BI app supports Outlook for the web and Outlook for Windows.

## Known limitations

In Outlook and Microsoft 365, the app has the following limitations:

- Viewing items in full screen option isn't supported yet.
- Deep links aren't supported yet in Outlook and Microsoft 365. Items open in a new browser tab.
