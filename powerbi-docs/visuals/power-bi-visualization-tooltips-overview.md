# Tooltips overview

[!INCLUDE [applies-yes-desktop-yes-service](../includes/applies-yes-desktop-yes-service.md)]

Tooltips display contextual information when consumers hover over data points or icons in a visual. They help users explore details without leaving the report. Power BI supports three types of tooltips, and you can use them together on the same visual.

This article gives a brief overview of each type and links to the detailed articles.

## Visual tooltips

Visual tooltips are the default tooltips that appear when a consumer hovers over a data point. They show the field values used in the visual, along with any extra fields you add to the **Tooltip** field well. You can format their text and background, and many visuals support drill actions in the tooltip footer.

For more information, see [Visual tooltips in Power BI](power-bi-visualization-visual-tooltips.md).

## Report page tooltips

Report page tooltips replace the default visual tooltip with a report page that you design. Use them when you want richer context, such as supporting visuals, KPIs, or images, that doesn't fit in a default hover card.

For more information, see [Create report page tooltips in Power BI](../create-reports/desktop-tooltips.md).

## Help tooltips

Help tooltips appear when a consumer selects the **Help tooltip** icon in the visual header. They display short text or a full report page that explains the visual or describes how to interact with it. Use them to guide consumers through a report.

For more information, see [Add Help tooltips to visuals in Power BI](power-bi-visualization-help-tooltips.md).
