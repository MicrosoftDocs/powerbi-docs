> [!TIP]
> For new solutions, we recommend [Dataflow Gen2](/fabric/data-factory/dataflows-gen2-overview), which offers improved performance, more destinations, and built-in AI. Existing Gen1 dataflows continue to work, but new feature development is focused on Dataflow Gen2. To learn about upgrading existing dataflows, see [Upgrade from Dataflow Gen1 to Dataflow Gen2](/fabric/data-factory/dataflow-gen2-migrate-from-dataflow-gen1).
