# Supported browsers for Power BI  

We support Power BI in these browsers:

-   Microsoft Edge

-   Internet Explorer 11

-   Internet Explorer 10

-   Chrome desktop latest version

-   Safari Mac latest version

-   Firefox desktop latest version
